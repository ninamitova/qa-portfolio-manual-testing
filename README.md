# QA Portfolio – Manual Testing

This repository contains my manual QA testing work for the demo e-commerce website **Demoblaze**.

## Project Overview
The goal of this project is to demonstrate my ability to:
- Analyze requirements
- Design test plans and test cases
- Execute manual tests
- Identify and report defects clearly

## Tools & Skills Used
- Manual Testing
- Test Case Design
- Bug Reporting
- SDLC & STLC
- GitHub & Markdown

## Project Structure
- **TestPlan/** – Test strategy, scope, and approach
- **TestCases/** – Functional test cases
- **BugReports/** – Defect reports with steps, severity, and expected vs actual results
